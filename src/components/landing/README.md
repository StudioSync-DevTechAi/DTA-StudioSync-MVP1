# Landing Page Visual Enhancements

This folder contains components and enhancements specifically for the landing page visual experience.

## Components

### DottedSurface
A Three.js-based animated dotted surface component that creates a dynamic 3D particle effect. Located in `src/components/ui/dotted-surface.tsx`.

## Usage

The DottedSurface component is used in the HeroSection to replace the static gradient overlay, providing a more dynamic and engaging visual experience.

## Future Enhancements

Additional landing page visual enhancements can be added to this folder as needed.
